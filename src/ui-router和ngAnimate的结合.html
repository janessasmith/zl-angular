1. 创建项目要有一个模板结构，需要布局文件、每个表单的视图文件、格式文件以及js文件，这个页面就是布局文件
           每一个表单的xxx.html表示层级结构中的html文件，这些结构最终创建了表单结构
        2. 布局文件需要加载我们所需的资源----AngularJS、ngAnimate、Ui Router以及其他脚本和样式表，并设定一个ui-view用来告知Ui Router我们的视图需要显示到哪里
        3. 在100.form-app.js文件中开始创建Angular应用和最基本的路由配置，也就是如何把Angular App应用到div上面
        4. 在100.form-app.js页面中，建立了相应的路由，给每个视图区域定义(url)，视图文件(templateUrl)和控制器(controller)
